# Multi-Client Chat Application

A desktop-based multi-client chat application built with Python, TCP sockets, multithreading, and Tkinter.

The application allows multiple users to connect to a central server, exchange public messages, send private messages, view online users, and disconnect safely.

## Features

- Multi-client support
- TCP socket communication
- Multithreaded server
- Tkinter graphical user interface
- Public group chat
- Private messaging
- Online-users panel
- Join and leave notifications
- Duplicate username protection
- Message timestamps
- Command handling
- Graceful disconnection
